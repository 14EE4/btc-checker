# BTC Checker - 비트코인 지갑 조회기

## 설명
비트코인 지갑 주소를 입력하여 현재 잔액과 자산 가치를 실시간으로 확인할 수 있는 웹 애플리케이션입니다.

## 기능
- ✅ 비트코인 지갑 주소 조회
- ✅ 보유 BTC 및 Satoshi 표시
- ✅ 실시간 USD/KRW 시세 조회
- ✅ 자산 총 가치 계산 (USD/KRW)
- ✅ 다크 모드 인터페이스

## API
- **mempool.space** - 비트코인 지갑 정보
- **CoinGecko API** - 비트코인 가격 정보

## 사용법
1. 비트코인 주소 입력 (bc1q..., 1A1z... 등)
2. 「조회」 버튼 클릭 또는 Enter 키 입력
3. 잔액과 자산 가치 확인
